사용법

  1. x.txt에 x축 값들을 넣는다.
  2. y.txt에 y축 값들을 넣는다.
  3. runGraphTool.bat을 더블클릭한다.

Filtering Parameter 변경

  1. GraphTool.py를 오른쪽 클릭
  2. 연결프로그램
  3. 메모장 클릭!
  4. y_filtered_with_gaussian = gaussianFilter(y, window_size=5, std=7)
    y_filtered_with_average = averageFilter(y,window_size=3) 이렇게 적힌 부분에서 숫자 있는 부분을 변경해봐
  5. window_size는 자기 자신의 값은 옆의 몇번째 데이터까지 보면서 영향을끼칠 것이냐는거고 std는 표준편차!
